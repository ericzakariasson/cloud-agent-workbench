# cloud-agent-workbench

## Pinball game

A self-contained browser pinball game lives in this repo.

### Run locally

1. From this folder, start a static server:
   - `python3 -m http.server 8000`
2. Open `http://localhost:8000` in your browser.
3. Play:
   - `Space` launches the ball
   - `Left/Right Arrow` (or `A/D`) control flippers
   - Click **Restart Game** to reset score and lives
