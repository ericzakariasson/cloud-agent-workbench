# Mini Pinball Game

A small browser-based pinball game built with plain HTML, CSS, and JavaScript.

Run locally:

1. From this folder, start a simple static server:
   python3 -m http.server 8000
2. Open:
   http://localhost:8000
3. Play:
   - Hold Space and release to launch the ball
   - Use A / L or Left / Right arrows for flippers
   - Press R after game over to restart
