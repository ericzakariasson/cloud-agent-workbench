# Retro Minesweeper

Simple no-build retro Minesweeper built with plain HTML/CSS/JS.

Run locally:

1. From the repo root, start a static server:
   `python3 -m http.server 8000`
2. Open `http://localhost:8000` in a browser.
3. Left click to reveal, right click to flag, and use "Reveal safe tile" for help.
