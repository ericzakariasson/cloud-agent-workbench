# Breakout / Arkanoid

Tiny self-contained Breakout clone built with HTML, CSS, and vanilla JavaScript.

## Run locally

Option 1 (quick): open `index.html` directly in your browser.

Option 2 (recommended static server):

```bash
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.

## Controls

- Mouse movement (on canvas): move paddle
- Keyboard: Left/Right arrows or A/D
- Space: launch ball when stuck, or toggle pause/resume
- Start button: start/pause/resume
- Reset button: restart the round
