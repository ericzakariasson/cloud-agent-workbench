# Retro Minesweeper

A small no-build retro Minesweeper game made with plain HTML, CSS, and JavaScript.

## Run locally

From this directory:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000` in a browser.

## How to play

- Left click to reveal a tile
- Right click to place/remove a flag
- Use `?` to reveal one safe tile when you are stuck
- Reveal every non-mine tile to win
